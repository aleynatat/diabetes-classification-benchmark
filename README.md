# diabetes-classification-benchmark

# Diabetes Classification & Missing Data Analysis

This project performs **Binary Classification** on the Pima Indians Diabetes Database.
It benchmarks various **Classification Algorithms** under two different data handling strategies.

## Models Used (Classifiers)
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Support Vector Classifier (SVC)
- Naive Bayes
- Decision Tree Classifier
- Random Forest Classifier
- AdaBoost Classifier
